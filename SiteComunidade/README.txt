=== GUIA DE REGRAS DO SISTEMA DE ANÁLISE DO CADASTRO AMBIENTAL RURAL - CAR ===

== Requerimentos:
- Nodejs


== Instalação:
- Entre na pasta do projeto e execute os seguintes comandos: 
  
  `sudo npm install`

  `sudo npm install -g grunt-cli`

  `grunt`

- Os arquivos do site serão compilados dentro da pasta 'dist/'. O conteúdo dessa pasta é que deve ser usado para fazer o deploy do site.
